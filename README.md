# Lista 2 de Exercícios Python
Este repositório contém a resolução de uma lista prática de 20 exercícios em Python. O objetivo deste projeto é consolidar os conhecimentos básicos e intermediários da linguagem, cobrindo desde operações simples com variáveis até a criação de funções e manipulação de estruturas de dados.

## Tópicos Abordados

Neste notebook, foram explorados e colocados em prática os seguintes conceitos de Python:

* **Manipulação de Strings:** Métodos como `.upper()`, `.lower()`, `.strip()` e formatação avançada (f-strings e `.format()`).
* **Estruturas de Dados:** * **Listas:** Criação, adição (`.append()`), remoção (`del`) e fatiamento (*slicing*).
  * **Tuplas:** Entendimento prático sobre imutabilidade.
  * **Dicionários:** Criação, acesso a valores por chaves, atualização e adição de novos pares chave-valor.
* **Estruturas Condicionais:** Uso de `if`, `elif` e `else` combinados com operadores lógicos (`and`, `or`) e matemáticos (como o operador de módulo `%`).
* **Estruturas de Repetição (Loops):**
  * Laços `for` aliados à função `range()` e para iteração de listas.
  * Laços `while` para contagens condicionais.
* **List Comprehension:** Sintaxe concisa para criação de listas baseadas em condições matemáticas.
* **Funções (`def`):** * Criação de funções com e sem parâmetros.
  * Funções com retorno condicional (`return`).
  * Cálculos matemáticos embutidos em escopo local.
